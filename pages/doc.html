What I did / Technologies I used

Meta Tags

The following meta tags were added to the head section of the HTML file:

<meta name="description" content="Exploring wearable devices, sensor technology, machine learning, and IoT in sport for motor learning and real-time biofeedback.">

<meta name="keywords" content="wearable devices, sport technology, motor learning, sensor technology, data processing, machine learning, IoT, biofeedback systems">

Sidebar Keyword List

Keywords were also added to a right-hand sidebar to give both users and search engines a quick overview of the main topics covered.

Structured Data (JSON-LD)

To improve discoverability, a <script type="application/ld+json"> block was added, providing search engines with additional information about the page:

{ "@context": "https://schema.org", "@type": "ScholarlyArticle", ... }

This enhances visibility in search results, especially for scholarly content.

3. Crosslinking

Which Websites to I link to

When building my site, I add links thoughtfully in different sections: on the description page at the beginning to give quick context, within keyword research sections to support related terms, and in the footer for general references or important external resources. The links shown on the site are just examples and don’t always perfectly fit every case. I always make sure to link only to relevant, trustworthy sources to keep the user experience smooth and meaningful.



Where and what to link

  • Key Words on the site bar
  • Other relevant and similar Papers

  • Relevant Topics and Recommendations in the footer
